# How to manage Account Level SSL certificates?
5centsCDN has released a new Account level SSL Certificates Management feature which will be a user-friendly feature in managing your SSL certificates. The main advantage of this feature is that you do not have to worry about uploading SSL certificates for each and every CDN resources instead, you can choose a certificate which needs to be uploaded only once and forget. Once this process is done with, you will get an option to use this installed SSL with your CDN zones. 
On the Dashboard click on "Settings"
