# How to enforce HTTPS using edge rules?
Using edge rule we can enable HSTS (HTTP Strict Transport Security) security feature that helps protect websites against protocol downgrade attacks and cookie hijacking by ensuring that browsers only interact with the site using HTTPS.
Go to your zone settings by clicking the "Manage" button
